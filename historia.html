📖 História do Projeto

O sistema de filmes foi desenvolvido com o objetivo de aplicar na prática os conceitos de HTML, utilizando páginas interligadas para simular um sistema de cadastro e consulta de dados.

🎯 Objetivo do Projeto

Criar um sistema simples com navegação entre páginas, contendo cadastro, consulta, login e organização de informações sobre filmes.

🛠️ Desenvolvimento

O projeto foi construído utilizando apenas HTML, com foco na estruturação de páginas, uso de formulários, tabelas e links de navegação entre as páginas do sistema.

👥 Divisão de Tarefas

Aluno Atividade Desenvolvida
Phelipe Criação das páginas Home e Login
Kaio Organização das páginas e estrutura geral do site
Julio Cadastro e Consulta de filmes (CRUD simulado)

📌 Conclusão

O projeto permitiu compreender melhor a criação de sites utilizando HTML, trabalhando com múltiplas páginas e organização de conteúdo.
