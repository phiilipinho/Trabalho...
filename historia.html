📖 História do Projeto

👥 Nosso Grupo

O Sistema de Filmes foi desenvolvido com o objetivo de colocar em prática os conhecimentos de HTML, criando um site com múltiplas páginas interligadas, simulando um sistema simples de gerenciamento de informações.

🎯 Objetivo do Projeto

O principal objetivo foi desenvolver um sistema web básico com navegação entre páginas, incluindo funcionalidades como cadastro, consulta e login, além de organizar informações sobre filmes de forma estruturada.

🛠️ Desenvolvimento

O projeto foi construído utilizando apenas HTML, com foco no aprendizado da estrutura de páginas web. Foram aplicados conceitos como formulários, tabelas, links de navegação e organização de conteúdo em seções bem definidas.

Julio Criação das páginas de Cadastro e Consulta (simulação de CRUD)

📌 Conclusão

A criação deste projeto contribuiu para o entendimento da estrutura de páginas web, permitindo a prática de organização de conteúdo, navegação entre arquivos e construção de um sistema simples utilizando HTML.
