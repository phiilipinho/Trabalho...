📖 História do Projeto

👥 Nosso Grupo

O Sistema de Filmes foi desenvolvido com o objetivo de colocar em prática os conhecimentos de HTML, criando um site com múltiplas páginas interligadas, simulando um sistema simples de gerenciamento de informações.

🎯 Objetivo do Projeto

O principal objetivo foi desenvolver um sistema web básico com navegação entre páginas, incluindo funcionalidades como cadastro, consulta e login, além de organizar informações sobre filmes.

🛠️ Desenvolvimento

O projeto foi construído utilizando apenas HTML, com foco no aprendizado da estrutura de páginas web. Foram utilizados formulários, tabelas, links de navegação e organização de conteúdo em seções.

Phelipe Desenvolvimento das páginas Home e Login

📌 Conclusão

Este projeto ajudou no entendimento da estrutura de páginas web, permitindo a prática de navegação entre arquivos e organização de conteúdo.
